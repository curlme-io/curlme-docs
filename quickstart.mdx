## Prerequisites

- Node.js 18+ installed
- A curlme.io account (sign up at [curlme.io](https://curlme.io/auth/signin))

## Step 1: Install the CLI

Install the curlme CLI globally using npm:

```bash
npm install -g @curlme/cli
```

Or using yarn:

```bash
yarn global add @curlme/cli
```

Verify the installation:

```bash
curlme --version
```

## Step 2: Authenticate

Generate an API key from your [account settings](https://curlme.io/account), then login:

```bash
curlme auth login
```

You'll be prompted to enter your API key:

```
? Enter API Key: cm_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
➜ Authenticated as you@example.com
```

## Step 3: Create Your First Bin

Create a new request bin to start capturing webhooks:

```bash
curlme bin create my-first-bin
```

```
➜ Bin created

  ID        abc123xyz
  Hook      https://curlme.io/h/abc123xyz
  Inspect   curlme listen
```

## Step 4: Start Listening

Open a terminal and start the real-time listener:

```bash
curlme listen
```

```
→ Listening on abc123xyz (active)
  Waiting for requests…

  [ Enter: inspect | R: replay | D: diff | O: open | Ctrl+C: quit ]
```

## Step 5: Send a Test Request

In another terminal, send a request to your bin:

```bash
curl -X POST https://curlme.io/h/abc123xyz \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello, curlme!"}'
```

You'll see the request appear in your listener:

```
→ POST   /                  200    45B  rq_f8a3b2
```

Press `Enter` to inspect the full request details.

## Step 6: Replay the Request

Replay the captured request to your local development server:

```bash
curlme replay --to http://localhost:3000
```

```
✔ Replayed rq_f8a3b2
────────────────────────────────────────
Target    http://localhost:3000/
Status    200
Duration  45ms
```
