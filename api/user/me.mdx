title: "Current User"

description: "Get the authenticated user's information"
---

## Request

<CodeGroup>
```bash cURL
curl https://curlme.io/api/user \
  -H "x-api-key: cm_your_api_key"
```

```javascript JavaScript
const response = await fetch('https://curlme.io/api/user', {
  headers: { 'x-api-key': 'cm_your_api_key' }
});
const user = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://curlme.io/api/user',
    headers={'x-api-key': 'cm_your_api_key'}
)
user = response.json()
```
</CodeGroup>

## Response

<ResponseField name="id" type="string">
  User ID
</ResponseField>

<ResponseField name="name" type="string">
  Display name
</ResponseField>

<ResponseField name="email" type="string">
  Email address
</ResponseField>

<ResponseField name="image" type="string">
  Profile image URL (from OAuth provider)
</ResponseField>

<ResponseField name="emailVerified" type="string">
  ISO timestamp of email verification (if applicable)
</ResponseField>

```json Example Response
{
  "id": "usr_abc123",
  "name": "John Doe",
  "email": "john@example.com",
  "image": "https://avatars.githubusercontent.com/u/12345",
  "emailVerified": null
}
```

## Delete Account

To delete your account and all associated data:

```bash
curl -X DELETE https://curlme.io/api/user \
  -H "x-api-key: cm_your_api_key"
```

<Warning>
  Account deletion is **permanent**. All data is removed:
  - All bins you own
  - All captured requests
  - API keys
  - Organization memberships
</Warning>

### Response

```json
{
  "success": true
}
```
